Question-2: What is the difference between innerHTML,innerText, and textContent ?

Ans:

      innerHtml returns all text, including html tags that is contained by an element.
      innerText returns all text contained by an element and all its child elements.
      textContents is all text contained by an element and all its children that are for formatting purposes only.

Question-3: What is event delegation in the DOM?

Ans: Event delegation in JavaScript is a pattern that efficiently handles events. Instead of adding events to each element events can be added to a parent element. It refers to the process of handling events at a higher level than the element from which the event originated, using event propagation bubbling.

Question-4: What is event bubbling in the DOM?

Ans: Event bubbling directs an event to its target. It works like this: When an element like a button is clicked an event is directed to the element. If an event handler is set for the element the event handler is triggered. Then the event bubbles up to the elements parent.
